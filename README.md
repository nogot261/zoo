# ZooCare Market

Учебный PHP-сайт зоомагазина с каталогом товаров, калькулятором корма, формой заявки и блоком животных, которые ищут дом.

## Состав

- `index.php` - главная страница.
- `catalog.php` - каталог товаров с поиском, фильтрацией и сортировкой.
- `main.css` - стили сайта.
- `data/products.php` - данные товаров.
- `data/adoptions.php` - данные животных.
- `images/` - изображения сайта, товаров и животных.
- `START_SITE.bat` - локальный запуск на Windows.
- `Dockerfile` - запуск через Docker и деплой на Render.

## Локальный запуск

Нужен PHP 8 или новее.

```powershell
php -S 127.0.0.1:8080
```

После запуска открыть:

```text
http://127.0.0.1:8080/index.php
```

На Windows можно запустить `START_SITE.bat`.

## Docker

```bash
docker build -t zoocare-market .
docker run --rm -p 8080:80 zoocare-market
```

После запуска открыть:

```text
http://127.0.0.1:8080/
```

## Render

Для Render подготовлен `Dockerfile` на базе `php:8.3-apache`.

Порядок деплоя:

1. Создать новый **Web Service**.
2. Подключить репозиторий `nogot261/zoo`.
3. В качестве окружения выбрать **Docker**.
4. Dockerfile path оставить `./Dockerfile`.
5. После деплоя открыть ссылку, которую выдаст Render.

Важно: заявки из формы сохраняются в `requests.csv` внутри контейнера. На бесплатном тарифе Render файловая система контейнера не предназначена для постоянного хранения данных, поэтому после пересборки или рестарта сохраненные заявки могут пропасть. Для постоянного проекта нужна база данных или внешнее хранилище.
